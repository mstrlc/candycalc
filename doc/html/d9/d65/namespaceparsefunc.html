Detailed Description

parsefunc is a module that handles input parsing for Candycalc

To use this module either use: parse_input(input) for parsing input to expression solve_expr(expr) to solve the expression get_result(input) to get a result of 2 functions above get_parsed_input(input) to get string representing the expression
